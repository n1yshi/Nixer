# Nixer Webpanel (Browser-only)

This folder contains the browser web UI + Node server, without the desktop app.

## Run

```bash
cd webpanel
npm install
npm start
```

Open: `http://localhost:10422`

## Config (optional)

- `PORT` (default: `10422`)
- `NIXER_NODE_DATA_DIR` (default: `~/.config/NixerNodeFull`)
